Points

Also known as dongers, dildos, bamboo, hdpoints

For every 10 minutes you spend watching the stream, you receive 1 point.
Points can be used for entering giveaways, gambling, joining whitelisted servers, and redeeming awards. You can read more about points here.

Bot Commands

!dongers Displays how many points you currently have
!gamble (amount) If bot rolls 1-60: lose bet, 61-98: 2x bet, 99-100: 3x bet
!info Displays more info about how to earn points
!leaderboard Links the leaderboard of the users with the most points
!rewards Links the info about earning rewards with points
!time Displays how much time you have spent in the stream
!uptime Displays how long Panda has been live for that stream
